## Велика підбірка витоків даних
Підбірка різноманітних витоків, наборів даних.     
Зібрано та упорядковано дослідниками KR. Labs Research.  
Публікується лише в цілях вивчення та ознайомлення, а також покращення кібербезпеки.  
Забороняється будь-яке несанкціоноване використання та поширення даних матеріалів.
